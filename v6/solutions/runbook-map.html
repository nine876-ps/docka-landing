Signal System

Four readiness levels per use-case.

Every runbook is graded on a four-level signal system so you can prioritize what matters.

L1 Unfunctional

Agents cannot complete this task. No runbook exists or the existing one produces consistent failures.

L2 Semi-functional

Agents sometimes complete this task but with inconsistent results. Runbook exists but has gaps.

L3 Fully-functional

Agents reliably complete this task. Runbook is complete and verified fail→pass.

L4 Optimized

Agents complete this task in one shot with minimal tokens. Runbook is optimized for cost and success rate.
